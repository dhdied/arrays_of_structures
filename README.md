# 📖 Описание задачи
Программа представляет собой систему управления базой данных из 20 горных вершин. Каждая запись содержит детальную информацию о вершине, организованную с помощью структур и перечислений.

---

# 🛠 Реализованный функционал
## Занятие 9: Структуры и алгоритмы
- Комбинированные типы данных: Использование вложенных структур (Location) и перечислений (enum) для типизации гор.
- Расчет статистики: Вычисление среднего значения высот всех вершин.
- Сортировка пузырьком: Рациональная перестановка всей структуры данных по возрастанию высоты.
- Анализ данных: Поиск стран местонахождения трех высочайших восьмитысячников из списка.
- Интерактивное редактирование: Поиск вершины по высоте и изменение её типа или страны.
- Динамическая память: Фильтрация гор по стране с копированием результатов в динамический массив (new / delete[]).
- Функции-обертки: Использование указателей на функции для реализации гибкого вывода отфильтрованных данных.

## Занятие 10: Работа с файлами (fstream)
Текстовая синхронизация:
- Экспорт данных в текстовый файл.
- Чтение данных из текстового файла с поиском совпадений по названию и обновлением высоты вершины в основном массиве.

Бинарная сериализация:
- Реализация подпрограммы для записи массива структур в бинарный файл.
- Восстановление (чтение) всей базы данных из бинарного файла напрямую в память.
