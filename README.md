# Проект «Горные вершины» (C++)

Учебная работа по дисциплине «Программирование», посвященная изучению структур данных, перечислений (enum), динамической памяти и алгоритмов сортировки в языке C++.

## 🏔 Описание задачи
Программа управляет базой данных из 20 горных вершин. Каждая запись содержит:
* Название вершины
* Высоту (в метрах)
* Страну местонахождения (вложенная структура)
* Тип горы (перечисляемый тип `enum`: вулканическая, складчатая и др.)

## 🛠 Реализованный функционал
В проекте реализованы следующие возможности:
1. **Расчет статистики:** вычисление среднего значения высот всех вершин.
2. **Сортировка:** рациональная сортировка массива структур по возрастанию высоты (метод «пузырька»).
3. **Фильтрация восьмитысячников:** поиск и вывод сведений о высочайших вершинах мира.
4. **Редактирование данных:** функция изменения типа и страны горы по её высоте.
5. **Динамическая память:** фильтрация гор по стране с помещением результатов в отдельный динамический массив.
6. **Функции-обертки:** использование указателей на функции для гибкого вывода данных.

## 🚀 Запуск проекта
Для компиляции и запуска вам понадобится любой компилятор C++ (например, GCC или Clang).

1. Скомпилируйте файл:
```bash
g++ structures.cpp -o structures
```
2. Запустите программу:

```Bash
./structures
```
